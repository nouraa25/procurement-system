## 🐛 Troubleshooting

### Problem: Login doesn't work
**Solution**:
- Verify email is exact (copy-paste)
- Password is: password123
- Check browser console for errors

### Problem: AI shows no suppliers
**Solution**:
- Select a category first
- Use "Electronics" or "Printing" (have suppliers)
- "Stationery" has no suppliers (will show empty)

### Problem: Chat doesn't appear
**Solution**:
- Accept or reject request first
- Chat only enabled after response
- Refresh page

### Problem: Language doesn't switch
**Solution**:
- Click language button
- Wait for page reload
- Clear localStorage if stuck
